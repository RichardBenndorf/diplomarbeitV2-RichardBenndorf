2.16. Beim Betrieb vorgegebene zulässige Massen (fakultativ): werden diese Massen angegeben, müssen sie nach Anhang IV der Richtlinie 97/27/EG überprüft werden: Intended registration/in service maximum permissible masses (optional), where these values are given, they shall be verified in accordance with the requirements of Annex IV to Directive 97/27/EC:

2.16.1. Beim Betrieb vorgegebene höchstzulässige Gesamtmasse (mehrere Angaben für verschiedene technische Konfigurationen möglich): Intended registration/in service maximum permissible laden mass (several entries possible for each technical configuration) (5): sie 2.8.

2.16.2. Beim Betrieb vorgegebene höchstzulässige Achslast je Achse oder Achsgruppe und bei Sattellanhängern und Zentralachshangern vorgegebene Stützlast nach Anhang des Herstellers, wenn diese niedriger als die technisch zulässible maximale Massen auf dem Anhangspunkt sind, in dem fall stets der technisch zulässible maximale Massen auf dem coupling point (several entries possible for each technical configuration) (5): sie 2.8.

2.16.3. Beim Betrieb vorgegebene höchstzulässige Massen je Achsgruppe (mehrere Angaben für verschiedene technische Konfigurationen möglich): Intended registration/in service maximum permissible mass on each axle group (several entries possible for each technical configuration) (5): sie 2.8.

2.16.4. Beim Betrieb vorgegebene höchstzulässige Anhängemasse (mehrere Angaben für verschiedene technische Konfigurationen möglich): Intended registration/in service maximum permissible towable mass (several entries possible for each technical configuration) (5): sie 2.11.
